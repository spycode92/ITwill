링크걸린 글자(...)는 커서가 손모양으로 바뀌고 밑줄이 생기며
색상은 3가지 중에서 나타남
초기링크=파랑 / 활성화링크=빨강 / 방문한링크=보라

상대경로로 링크 걸기

  1. 웹문서, html문서 (*.htm/*.html)
    이미지 수업 보기 목록 수업 보기
  2. 이미지file (*.jpg/*.gif/*.png/...)
    이미지 움짤
  3. 동영상file (*.ogg/*.webm/*.mp4)
    동영상flower.mp4 동영상green.mp4
  4. text file (*.txt)
    memo

웹브라우저가 처리 불가능한 종류를 링크 걸어 놓으면 img.zip ( 다운로드 ) 된다

링크 여는 방식 지정 - target=" "

  • _self = 기본값. 현재 위치에서 열기
  • _blank = 새 탭에서 열기
    naver
  • _top = (프레임구조에서) 나뉨없이 전체에 열기
  • _parent = (프레임구조에서) 상위창에 열기
